# MediaPlayer-VolumeControl
调节音量。

## Import
[CocoaPods](http://cocoapods.org)

`pod 'MediaPlayer-VolumeControl', '~> 1.0.0'`

## Usage
```objective-c

#import "VDMediaVolume.h"

[VDMediaVolume currentVolume];
[VDMediaVolume setVolume:0.5f];
[VDMediaVolume raiseVolume:0.1f];

```
